# Card

卡片组件

### 示例

<m-crad img-src="/public/logo.png" summary="test"/>

### 代码
```html
<m-crad img-src="/public/logo.png" summary="test"/>
```

### Attributes
| 参数 | 说明 | 类型 | 是否必要 | 默认值 |
| ---- | ---- | ---- |   ----  |  ----  |
| width | 卡片高度 | Number | false | -|
| imgSrc | 图片资源地址 | String | true | -|
| imgHeight | 图片高度 | Number | false | -|
| summary | 卡片概要 | String/Slot | false | -|
| footer | 卡片底部 | Solt | false | -|
